# exit4067
Библиотека для работы с аналоговым 16-канальным мультиплексором на базе hp4067
# Что такое hp4067
Это аналоговый мультиплексор. Он позволяет соединять SIG с любым из 16 каналов как на вход (расширитель АЦП), так и на выход (расширитель ЦАП/ШИМ).
 > [!NOTE]
> Все остальные каналы режиме расширителя ЦАП, кроме выбранного, просто никуда не подсоединены, т. е. ШИМ/аналоговый сигнал на них не идёт

# Зависимости
Не имеются

# Подключение
 - VCC - от 2 до 6 вольт (напр. пин VCC)
 - S1 - S4 - любые цифровые пины
 - SIG - любой аналоговый пин для входа, ЦАП или ШИМ пин для выхода в зависимости от платформы
 - EN - высокиий уровень для отключения от всех каналов. Можно не подключать

![схема подключения](https://github.com/karasExit/exit4067/blob/main/photos/hp4067.png)
# Инициализация
```
exit4067 ...(); // Analog можно не указывать - но метод READ будет возвращать 0, write не будет работать
```
# Методы
```
switchReg(uint8_t reg) // переключить канал
read(uint8_t reg) // switchReg + прочитать через analogRead (канал > 15 - будет 0, analog не указан - тоже 0)
write(uint8_t reg, int duty) // switchReg + analogWrite (канал > 15 - действия нет, analog не указан - тоже нет действия)
```
